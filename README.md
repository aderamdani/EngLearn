# EngLearn

**Interactive macOS English Learning Application**

[![Latest Release](https://img.shields.io/github/v/release/aderamdani/EngLearn)](https://github.com/aderamdani/EngLearn/releases)
[![Platform](https://img.shields.io/badge/platform-macOS%2026+-blue)](https://developer.apple.com/macos/)
[![Language](https://img.shields.io/badge/Swift-6-orange)](https://swift.org)
[![Architecture](https://img.shields.io/badge/arch-ARM64-green)](https://support.apple.com/en-us/116943)
[![License](https://img.shields.io/badge/license-MIT-lightgrey)](LICENSE)

**EngLearn** is a comprehensive, handcrafted macOS application designed to take English learners from Beginner (A1) to Proficiency (C2). Built with SwiftUI, SwiftData, and Liquid Glass. Curriculum inspired by Oxford, Cambridge, BBC Learning English, and the British Council. All explanations in Bahasa Indonesia. Zero third-party dependencies.

---

## Core Modules

| Module | Description |
| --- | --- |
| **Grammar Lab** | Interactive lessons with multiple choice, fill-in-the-blank, and reorder exercises |
| **Vocabulary Forge** | 3D flashcards with SM-2 spaced repetition for optimal retention |
| **Reading Room** | Graded passages with comprehension quizzes |
| **Listening Lab** | TTS playback, dictation exercises, and comprehension activities |
| **Writing Workshop** | Guided prompts with word count and sample answer comparison |
| **Speaking Studio** | 44 phoneme pronunciation guide with speech recognition feedback |
| **Daily Lessons** | 5-10 minute structured daily lessons with gamification |
| **Immersion Zone** | Authentic content, idioms, and real-world English context |

## Technical Excellence

- **Swift 6**: Strict concurrency safety, modern language features
- **SwiftUI**: Native macOS interface with Liquid Glass effects
- **SwiftData**: Local persistence for progress, vocabulary, and writing
- **ARM64 Only**: Optimized exclusively for Apple Silicon (M1/M2/M3/M4+)
- **macOS 26+**: Built for the latest macOS design language
- **Zero Dependencies**: Pure Apple frameworks, no third-party libraries
- **Performance**: <3% idle CPU, <150MB memory, <2s launch

## Installation

1. Visit the **[Releases](https://github.com/aderamdani/EngLearn/releases)** page
2. Download the latest `EngLearn-vX.X.X.dmg`
3. Open the DMG and drag **EngLearn** to your `Applications` folder

Requires macOS 26+ on Apple Silicon Mac.

## Bahasa Indonesia

**EngLearn** adalah aplikasi macOS yang dirancang untuk membawa pelajar bahasa Inggris dari level Pemula (A1) hingga Mahir (C2). Dibangun dengan SwiftUI, SwiftData, dan Liquid Glass. Kurikulum terinspirasi dari Oxford, Cambridge, BBC Learning English, dan British Council. Semua penjelasan dalam Bahasa Indonesia. Tanpa dependensi pihak ketiga.

### Fitur Utama

- **Tata Bahasa**: Pelajaran interaktif dengan pilihan ganda, isian, dan susun kalimat
- **Kosakata**: Kartu flash 3D dengan sistem pengulangan berjarak SM-2
- **Membaca**: Teks bacaan berjenjang dengan kuis pemahaman
- **Mendengar**: Audio TTS, latihan dikte, dan aktivitas pemahaman
- **Menulis**: Latihan terpandu dengan hitungan kata dan perbandingan jawaban contoh
- **Berbicara**: Panduan pengucapan 44 fonem dengan umpan balik pengenalan suara
- **Pelajaran Harian**: Pelajaran terstruktur 5-10 menit per hari
- **Zona Imersi**: Konten autentik, idiom, dan konteks bahasa Inggris dunia nyata

### Keunggulan Teknis

Dibangun 100% menggunakan **Swift 6 & SwiftUI** tanpa framework pihak ketiga. Hanya untuk Apple Silicon (ARM64). CPU idle di bawah 3%, memori di bawah 150MB, waktu buka kurang dari 2 detik.
